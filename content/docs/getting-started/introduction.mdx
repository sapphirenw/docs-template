Welcome to the documentation! This platform is built to make creating beautiful, interactive documentation simple and enjoyable.

## What is this?

This is a modern documentation platform inspired by [Mintlify](https://mintlify.com) and [GitBook](https://gitbook.com). It's built with Next.js and MDX, offering a powerful yet simple way to create stunning documentation.

<Tip>
  This entire documentation site is built using the platform you're learning about. How meta!
</Tip>

## Key Features

- **MDX Support** - Write Markdown and use React components
- **Dark Mode** - Beautiful dark and light themes
- **Search** - Lightning-fast search with keyboard shortcuts
- **Responsive** - Looks great on all devices
- **Components** - Rich set of documentation components

## Quick Start

Getting started is easy:

<Steps>
  <Step title="Install the CLI">
    First, install the documentation platform CLI globally.

    ```bash
    npm install -g docs-platform
    ```
  </Step>

  <Step title="Initialize your project">
    Create a new documentation project in your desired directory.

    ```bash
    docs-platform init my-docs
    cd my-docs
    ```
  </Step>

  <Step title="Start writing">
    Open your editor and start writing documentation in the `content/docs` folder.
  </Step>

  <Step title="Preview locally">
    Run the development server to preview your changes.

    ```bash
    npm run dev
    ```
  </Step>
</Steps>
